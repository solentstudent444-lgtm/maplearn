# maplearn
🌍 MapLearn  A full-stack e-learning platform built with Next.js and Firebase, enabling students to enroll in courses, track progress, and explore course locations using an interactive map interface.

🚀 Live Demo

(pon aquí tu Vercel URL)

🧠 Features
🔐 Authentication (Firebase Auth)
👥 Role-based access (Student / Admin)
📚 Course enrollment system
📊 Progress tracking
🗺️ Interactive map (Leaflet)
🛠️ Admin dashboard
📱 Fully responsive UI
🧱 Tech Stack

Next.js (App Router)
React
Firebase (Auth + Firestore)
Leaflet Maps
Tailwind CSS

⚙️ Installation
git clone https://github.com/yourrepo/maplearn
cd maplearn
npm install
npm run dev

🔐 Environment Variables

Create .env.local:

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=

🧪 Testing & Quality
npm run lint
npm run build
npm run start
